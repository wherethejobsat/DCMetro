# Launch Post

## Short version

I made a small DC Metro exit guide: https://wherethejobsat.github.io/DCMetro/

Pick a station, line, and direction. It tells you the train car and door closest to escalators, stairs, elevators, and other exits. It works offline after first load, because station signal is not a thing I trust.

No live outage data. Not affiliated with WMATA. If a station result looks wrong, please open a correction issue. If it saves you a platform hike, a GitHub star helps other riders find it.

## Slightly longer version

I built a small DC Metro exit guide for the part of the trip where you are already on the platform and just want to board near the right exit.

The app lets you choose a station, line, and direction, then shows the closest train car and door for escalators, stairs, elevators, and other egress points. It is a static GitHub Pages site, works offline after first load, and has no account, ads, tracking, backend, or runtime WMATA API dependency.

It does not include live elevator or escalator outage status, and it is not affiliated with WMATA. Station layouts and access conditions can change, so corrections are welcome. Please include the station, line, direction, current result, corrected result, and whatever source or evidence you have.

App: https://wherethejobsat.github.io/DCMetro/
Repo: https://github.com/wherethejobsat/DCMetro
